# noonejf.github.io

Portafolio personal de **José Fabián Sifuentes Gálvez** — Ingeniero Mecatrónico (PUCP).

🌐 **Live:** https://noonejf.github.io

Sitio estático (HTML/CSS/JS puro, sin frameworks ni build). Bilingüe ES/EN con
toggle persistente. Los CVs en PDF se generan con LaTeX (tectonic) desde
`PROYECTOS/DATA_ME` y se copian a `cv/`.

## Estructura

```
index.html   — todo el sitio (estilos y JS embebidos)
cv/          — CVs en PDF (completo / electrónico / programador × ES / EN)
```

## Deploy

Es un repo de GitHub Pages de usuario: todo lo que esté en `main` se publica
automáticamente en `https://noonejf.github.io`.

```powershell
git add .
git commit -m "update portfolio"
git push
```
